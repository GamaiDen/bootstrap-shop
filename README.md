# Bootstrap Shop

Проект для домашнего задания по вёрстке.

## Страницы

- **Главная** (`index.html`) — приветствие и карточки товаров
- **Каталог** (`catalog.html`) — сетка товаров
- **Категории** (`category.html`) — категории товаров
- **Контакты** (`contacts.html`) — форма обратной связи

## Технологии

- HTML5
- Bootstrap 5 (CDN)
- Python `http.server`

## Запуск

```bash
python3 app.py
